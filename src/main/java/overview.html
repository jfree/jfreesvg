JFreeSVG is a vector graphics library for the Java(tm) platform that allows Java2D drawings to be rendered to Scalable Vector Graphics (SVG). The key class in this library is:

  • the {@link org.jfree.graphics2d.svg.SVGGraphics2D} class for generating output in Scalable Vector Graphics (SVG) format;

    • In addition, there is an experimental {@link org.jfree.graphics2d.canvas.CanvasGraphics2D} class for translating Java2D calls to Javascript HTML5 Canvas API calls.

    JFreeSVG requires Java 6 or later. For more information, see http://www.jfree.org/jfreesvg/index.html.
